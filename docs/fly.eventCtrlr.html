Class: eventCtrlr

fly. eventCtrlr

The Event Controller

eventCtrlr is the main pub/sub object, paper events all publish through it, and objects can subscribe to events which are handled through their eventCall method.

new eventCtrlr()

The event controller is constructed when you call fly.init(). All mouse and key events are handled with paper tools and implemented within flypaper.

On-frame events must be initaited in the main javascript on window load.

Example
    // The eventCtrlr is created on init
    fly.init();
    // The frame events must be created on window load
    paper.view.onFrame = function(event) {
       fly.eventCtrlr.publish("frame");
    };
