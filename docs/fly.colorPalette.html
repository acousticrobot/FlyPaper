Class: colorPalette

fly. colorPalette

Populates fly.color with a colorset.

new colorPalette(args)

Returns the name of the current color palette if no args are passed in. Arguments passed on fly.init will be passed to the colorPalette. Args can be a string matched against predefined sets of colors in colorSets. Args can also be an object containing a name and a color set. If the set is legit, it will be added to the colorSets array, if the name exists already, the new set will replace the old.

Parameters:
Name Type Description
args String, Object

Example
fly.init({colorPalette:"neon"})
// sets the predefined neon color set on init

fly.colorPalette("pastel")
// changes to the predefined "pastel" color set

fly.colorPalette({
     name: "my color set", set: [
         ['red','#F04510','#FF7070','#FFD3C0'],
         ['orange','#F28614','#FFB444','#FFE8C0'],
         ['yellow','#CDB211','#FFFF70','#FFFFC0'],
         ['green','#42622D','#89C234','#C0FFC0'],
         ['blue','#00597C','#00A9EB','#B0E5FF'],
         ['purple','#6F006F','#9F3DBF','#FFC0FF'],
         ['grey','#383633','#A7A097','#FFFFFF']
     ]
})
// Adds "my color set" to the color sets, and
// redefines fly.colors to the custom color palette.

fly.colorPalette()
// returns "my color set"
