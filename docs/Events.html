Mixin: Grant Events

Grant Events

Gives any object all the necessary functions to register and receive events. Objects can subscribe to events through the Event Controller.

Granting Events

Objects inheriting from fly.base already have been granted event methods. For other objects, call fly.grantEvents(myObject);

Register Event

To register for events, use registerEvent()

Deregister Event

To deregister from events, use deregisterEvent()

Event Call

eventCall(event,args) is the method called by Event Controller for registered events. See eventCall()

Log Events

To log registered events to string, use logEvents()
