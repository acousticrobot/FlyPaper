Class: layers

fly. layers

The object responsible for containing the drawing layers. By default, there will be a background layer for color or animations, the main drawing layer, and a top layer for the info controller.

new layers(layers, background)

The Stage layers are created on fly.init. Defaults to "background", "layer 1", and "info"

reserved words

"background" and "info" are reserved layer names, background is primarily used for a solid color. See fly.colorUtil.background for setting background color.

Parameters:
Name Type Description
layers Integer | Array

Defaults to 1. Integer values will create that many layers, plus the background and info layers, layers will be named "layer 1", "layer 2" etc. Array of names will create that many layers, each can be referanced by that name.
background Boolean

Defaults true. If set to false, no background layer is created, first layer by integer is named "layer 0"
